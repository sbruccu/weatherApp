# weatherApp

## What I need ?
- Python 3

## How I run the app ?
- Clone the repository
- Open a terminal and move in the wheaterApp directory
- Activate the venv
```
source bin/activate
```    
- Run app.py
```
python3 app.py
```
- Open the localhost link in a browser

## How the app works ?
- In the input box type a city
- Click the button *Get weather* or type *Enter*
